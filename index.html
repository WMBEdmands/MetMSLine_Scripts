Getting Started: MetMSLine Instructions

Purpose

MetMSLine and associated software functions were created in the R language to attempt to achieve improved standard operating procedures and good laboratory practice for efficient and objective untargeted metabolomics analysis of large-scale (e.g. >300 samples) high-resolution LC-MS datasets.

You will need to install the latest version of R (available here: R.download.page) and also following R installation RStudio is recommended for its user-friendly interface (available here: Rstudio)

Required packages

-Initially install all R package dependencies by typing or pasting these lines of code into the command line of the R or RStudio console, you will only need to do this once per computer operating system:

install.packages(c("fgui","tcltk2","zoo", "gplots", "ggplot2", "sfsmisc", "RColorBrewer", "readMzXmlData"))
source("http://bioconductor.org/biocLite.R")
biocLite("pcaMethods")

What does it do?

MetMSLine is compatible with any peak picking software output (e.g. XCMS, mzMine and proprietary softwares) and is available as both a user friendly graphical user interface for non R specialists and also as command line functions for seamless integration. It is possible to use MetMSLine on any operating system.

Simply download the .zip file in the top right corner of this page, extract and save the files within a local directory on your computer (e.g. D:/MetMSLine).

In order to commence the MetMSLine processing sequence with a GUI simply type the following example commands into your R or RStudio (recommended) console e.g.

source('D:/MetMSLine/PreProc.QC.RLSC.GUI.R')
###This will start the MetMSLine GUI for pre-processing.###

Following pre-processing type/paste the following commands in this sequence to continue the data processing sequence.

source('D:/MetMSLine/Auto.PCA.GUI.R')
source('D:/MetMSLine/Auto.MV.Regress.GUI.R')
source('D:/MetMSLine/Auto.MS.MS.match.GUI.R')
source('D:/MetMSLine/DBAnnotate.GUI.R')

The MetMSLine installation also includes synthetic high resolution LC-MS data (5000 MS features, 80 samples, 20 Quality Controls) and synthetic continuous Y-variable data (10 Y-variables) with default settings.
